# MQTT-JS-TRAFFIC-CONTROLLER
MQTT JavaScript NHL Hogeschool Wrapper to connect to MQTT brokers and send/recieve data.

Made by Niels Visbeek & John Hawkins for Software Development 2019

# Installation
- Clone package
- ``npm install``
- ``cd ./app``
- ``node main.js``

# Building for web
- ``npm install``
- ``npm run build``
